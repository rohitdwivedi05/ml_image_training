# ML Image Training

This repository contains a **machine learning image training project** built using TensorFlow and Keras.  
It demonstrates how to build and train a **Convolutional Neural Network (CNN)** for image classification tasks, including data loading, preprocessing, augmentation, model training, evaluation, and visualization.

---

## 🚀 Project Overview

This project is focused on training a deep learning model to learn from image datasets. The goal is to:

- Load and preprocess image data
- Augment training images to improve generalization
- Build a CNN model using TensorFlow/Keras
- Train and evaluate the model on labeled image data
- Visualize performance metrics

The main code and experimentation are contained in the `Untitled0.ipynb` notebook.
